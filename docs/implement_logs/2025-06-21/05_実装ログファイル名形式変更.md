### 3. 変更方法
Bashスクリプトを使用して一括変更：
```bash
for f in *.md; do 
  new=$(echo "$f" | sed 's/^\([0-9]\)_/0\1_/')
  if [ "$f" != "$new" ]; then 
    mv "$f" "$new"
  fi
done
```
