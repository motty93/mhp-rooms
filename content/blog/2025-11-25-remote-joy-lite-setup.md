## はじめに

Remote Joy Liteは、PSPの画面をリアルタイムでPCに表示できるプラグインです。モンスターハンターポータブルシリーズを大画面でプレイしたり、プレイ動画を録画したい方におすすめのツールです。

この記事では、Remote Joy Liteの導入方法から使い方まで、初心者の方でもわかりやすく解説します。

## Remote Joy Liteでできること

- PSP画面のPC表示（リアルタイムストリーミング）
- ゲームプレイの録画（AVI形式）
- スクリーンショット撮影（BMP形式）
- PCキーボードでのPSP操作
- 全画面表示・アスペクト比調整

## 必要な環境

### PSP側

- **カスタムファームウェア（CFW）導入済みのPSP**（必須）
- 対応機種：PSP-1000 / PSP-2000 / PSP-3000 / PSP Go
- メモリースティック（PSP Goは内蔵ストレージ）

### PC側

- OS：Windows XP / Vista / 7 / 8 / 10 / 11
- PSP Type Bドライバ
- USBケーブル（PSP接続用）

### 重要な注意事項

**カスタムファームウェア（CFW）の使用は自己責任です。**公式ファームウェアでは動作しません。CFWの導入方法については別の記事で詳しく解説します。

## 導入手順

### ステップ1：ダウンロード

必要なファイルをダウンロードします。

- Remote Joy Lite本体（v0.19またはv0.20α）
- libusb-win32ドライバ（`libusb-win32-bin-1.2.6.0.zip`）

### ステップ2：PSP側のセットアップ

#### プラグインファイルの配置

1. PSPをUSBモードでPCに接続
2. メモリースティックのルートに`seplugins`フォルダを作成（既に存在する場合はスキップ）
3. `RemoteJoyLite.prx`を`ms0:/seplugins/`にコピー

PSP Goの場合は`ef0:/seplugins/`になります。

#### 設定ファイルの編集

`seplugins`フォルダ内の以下のテキストファイルに設定を追記します。

- `game.txt`：PSPゲーム・自作アプリで使用
- `vsh.txt`：XMB（メニュー画面）で使用

各ファイルに以下の行を追加します。

```
ms0:/seplugins/RemoteJoyLite.prx 1
```

PSP Goの場合は以下のように記述します。

```
ef0:/seplugins/RemoteJoyLite.prx 1
```

#### プラグインの有効化

1. PSPをリカバリーモード（R押しながら起動）で起動
2. Plugins → 各プラグインを`Enabled`に設定
3. PSPを再起動

### ステップ3：PC側のセットアップ

#### PSP Type Bドライバのインストール

1. `libusb-win32-bin-1.2.6.0.zip`を解凍
2. `inf-wizard.exe`を**管理者として実行**
3. 「PSP Type B」を選択
4. 画面の指示に従ってドライバをインストール
5. デバイスマネージャーで「PSP Type B」が認識されていることを確認

**Windows 10/11での注意点**

デバイスマネージャーで「PSP Type B」が認識されない場合は、以下を試してください。

- UEFI設定でセキュアブートを無効化
- ドライバ署名の強制を無効化（詳細スタートアップオプション）
- Zadigツールで「libusb-win32」ドライバに変更

## 使い方

### 基本操作

1. PSPとPCをUSBケーブルで接続
2. PC側で`RemoteJoyLite.exe`を起動
3. PSP画面がPC上に表示されます

### キーボードショートカット

- `Alt + Enter`：全画面表示の切り替え
- `Esc`または右クリック：設定メニュー表示
- `F1`：FPS・カラーモード表示
- `F3`：画面転送のON/OFF
- `F4`：ウィンドウタイトルバー表示切替

### 推奨設定

設定画面の「Etc」タブで「アスペクト比を保持する」にチェックを入れることをおすすめします。

処理落ちが気になる場合は、FPSを60から30または10に下げると改善することがあります。

### 音声出力

Remote Joy Lite単体では音声転送機能がありません。音声を出力するには、PSPのイヤホン端子とPCのライン入力端子をオーディオケーブル（別売り）で物理接続する必要があります。

## モンハンでの活用法

### 大画面プレイ

PSPの小さな画面では見づらかったモンスターの動きも、PC画面なら細部まで確認できます。特にMHP2GやMHP3での狩りがより楽しくなります。

### プレイ動画の録画

Remote Joy Liteには録画機能が搭載されています。自分の狩りの様子を録画して、後から見返したり、友人と共有したりできます。

### 配信での活用

配信ソフト（OBSなど）と組み合わせることで、PSPのゲーム画面を配信することも可能です。

### 遅延について

リアルタイムストリーミングのため、理論上は多少の遅延が発生します。ただし、実際に使用してみた感想としては、**モンハンのプレイには特に問題を感じませんでした**。アクション性の高いゲームでも快適にプレイできています。

## トラブルシューティング

### 「WAITING」表示のまま画面が映らない

最も多い原因は、PSP Type Bドライバの問題です。

**対処法**

1. デバイスマネージャーで「PSP Type B」が認識されているか確認
2. ドライバを一度削除し、再インストール
3. 管理者権限で`inf-wizard.exe`を実行
4. Zadigツールで「libusb-win32」に変更してみる

### プラグインが有効にならない

**確認事項**

- `RemoteJoyLite.prx`が正しいフォルダに配置されているか
- txtファイルの記述が正しいか（パスと末尾の「1」）
- リカバリーモードでプラグインが`Enabled`になっているか

### Windows 10/11で動作しない

- UEFI設定でセキュアブートを無効化
- ドライバ署名の強制を無効化（詳細スタートアップオプション）

### 録画動画にノイズが入る

- 別のUSBケーブルを試す
- PC側のUSBポートを変更してみる

### ゲームによっては動作しない

- CFWバージョンを6.35 PRO-B2以降に更新
- 拡張メモリ使用プラグインとの競合を確認し、不要なプラグインを無効化

### 音声が出ない

Remote Joy Lite単体では音声転送機能がありません。PSPとPCをオーディオケーブル（別売り）で物理接続する必要があります。

## メリット・デメリット

### メリット

- PSP画面を大画面で楽しめる
- 録画・配信が簡単にできる
- 設定が比較的シンプル
- 無料で利用できる

### デメリット

- CFW必須（自己責任）
- 音声は別途ケーブル接続が必要
- 一部のゲームでは互換性問題がある
- ドライバ設定がやや複雑（特にWindows 10/11）

## 代替手段

遅延が気になる場合や、より高画質な映像を求める場合は、以下の方法も検討してみてください。

- コンポーネントケーブルでのTV出力
- キャプチャーボードを使用した録画
- エミュレータ（PPSSPP）での高解像度プレイ

## まとめ

Remote Joy Liteは、PSPのゲーム画面をPCで楽しむための優れたツールです。特に録画や配信を目的とする場合には非常に便利です。

導入にはCFWが必要で、ドライバ設定にやや手間がかかりますが、一度設定してしまえば快適に使用できます。モンハンを大画面でプレイしたい方は、ぜひ試してみてください。

自己責任にはなりますが、適切に設定すれば実用上問題なく使用できます。この記事が皆さんのPSPライフをより楽しくする手助けになれば幸いです。
