## <a id="overview"></a>1. HuntersHubとは

HuntersHubは、モンスターハンターシリーズのオンラインプレイで仲間を募集するための情報共有サービスです。
部屋情報を作成してSNSなどで共有することで、一緒にプレイする仲間を簡単に見つけることができます。

> **注意：** HuntersHub自体はオンラインプレイ機能を提供していません。実際のオンラインプレイには、各プラットフォームに対応した通信ツールが必要です。

---

## <a id="register"></a>2. アカウント登録

部屋の作成や参加にはアカウント登録が必要です。以下の手順で登録してください。

### 登録手順

1. **新規登録ページへアクセス**
   ヘッダーの「新規登録」ボタンをクリックします。

2. **Google認証**
   「Googleで登録」ボタンをクリックし、Googleアカウントで認証します。

3. **利用規約への同意**
   利用規約とプライバシーポリシーを確認し、同意のチェックを入れます。

4. **登録完了**
   「登録する」ボタンをクリックすると、登録が完了します。

## <a id="create-room"></a>3. 部屋を作成する

オンラインプレイの準備ができたら、HuntersHubで部屋情報を作成しましょう。

### 作成手順

1. **部屋作成ページへ**
   ログイン後、「部屋を作る」ボタンをクリックします。

2. **ゲームバージョンを選択**
   プレイするゲーム（MHP、MHP2、MHP2G、MHP3など）を選択します。

3. **部屋情報を入力**
   - 部屋名（例：「初心者歓迎！楽しく狩ろう」）
   - 詳細説明（クエスト内容、募集条件など）
   - 最大人数（2〜4人）
   - パスワード（任意）

4. **部屋を作成**
   「作成する」ボタンをクリックすると、部屋コードが発行されます。

> **ヒント：** 部屋名や説明は、どんなプレイをしたいかが分かりやすく書くと、仲間が集まりやすくなります。

## <a id="join-room"></a>4. 部屋に参加する

他のプレイヤーが作成した部屋に参加する方法を説明します。

### 参加方法

1. **部屋を探す**
   「部屋一覧」から参加したい部屋を探すか、共有された部屋コード/URLからアクセスします。

2. **部屋の詳細を確認**
   部屋の説明、現在の参加人数、募集条件などを確認します。

3. **参加する**
   「参加する」ボタンをクリックします。パスワードが設定されている場合は入力が必要です。

4. **オンラインサービスで合流**
   実際のゲームでオンラインサービスに接続し、ホストの部屋に参加します。

---

## <a id="share"></a>5. 部屋情報を共有する

作成した部屋情報を友達やSNSで共有して、仲間を集めましょう。

### 共有方法

#### 部屋コードで共有
6文字の部屋コード（例：ABC123）を友達に伝えます。
HuntersHubの検索機能で部屋コードを入力すると、すぐに部屋が見つかります。

#### URLで共有
部屋の詳細ページのURLをコピーして、SNSやメッセージアプリで共有します。
URLをクリックすると直接部屋ページにアクセスできます。

#### SNS共有ボタン
部屋詳細ページの共有ボタンから、X（Twitter）などに簡単に投稿できます。

## <a id="online-play"></a>6. オンラインプレイの準備

HuntersHubで部屋を作成・参加した後、実際にオンラインプレイを始めるための準備について説明します。

### ポータブルシリーズ（PSP）の場合

PSP版モンスターハンターポータブルシリーズ（MHP、MHP2、MHP2G、MHP3）でオンラインプレイをするには、以下のいずれかの方法を使用します：

- **PS3のアドホックパーティ**
  PS3とPSPを接続して、インターネット経由でアドホック通信を行います。

- **XLink Kai**
  PCを使用してアドホック通信をトンネリングするソフトウェアです。

### MHXX（3DS/Switch）の場合

MHXX（3DS版/Switch版）でオンラインプレイをするには、以下の方法を使用します：

- **Pretendo Network**
  3DS/Switch向けのコミュニティネットワークサービスです。

## <a id="tips"></a>7. 便利な使い方・ヒント

### 効果的な部屋作成のコツ

- **分かりやすい部屋名**
  「HR3↑ リオレウス討伐」など、目的が明確な部屋名にしましょう。

- **詳細な説明**
  装備の指定、プレイスタイル、ボイスチャットの有無など、詳しく書きましょう。

- **適切な募集人数**
  クエストの難易度に応じて、適切な人数を設定しましょう。

### マナーとエチケット

- 参加前に部屋の説明をよく読みましょう
- 募集条件を満たしているか確認しましょう
- 退出時は一言挨拶をしましょう
- 楽しくプレイすることを心がけましょう

### トラブルシューティング

- **部屋が見つからない**
  部屋コードが正しいか確認し、部屋がまだアクティブか確認してください。

- **参加できない**
  部屋が満員になっていないか、パスワードが正しいか確認してください。

- **接続できない**
  オンラインサービスの設定を確認し、ネットワーク接続を確認してください。
